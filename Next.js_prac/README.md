# Next.js_prac

#### **`week26-1`** : Next.js를 활용한 SSR 실습
#### **`week26-2`** : Next.js 13를 활용한 실습

## 과제하는 법
### 1. git clone 하기

```bash
git clone https://github.com/jungsunbeen/Next.js_prac
cd week26-2
npm install
```

### 2. pocketbase 설정하기

**exe파일은 꼭 week26-2 파일에 넣어주어야합니닷**

`./pocketbase serve`

pocketbase에 대한 내용은 세션 자료에 있습니다. (세션 때 진행한 거 복붙해도 작동함 !)

### 3. 파일 실행하기

`npm run dev`
